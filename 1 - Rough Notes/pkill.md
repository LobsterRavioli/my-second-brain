# pkill

## Syntax
```bash
pkill [process name]
```

## Description
Kill a process by his name.
## Option

| Option | functionallity |
| ------ | -------------- |
|        |                |


## Examples
- kill process names sleep.
```bash
$ pkill sleep
[1]+  Terminated              sleep 60

```
