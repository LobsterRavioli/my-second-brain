# cp

Utility for copying file from a location to another.

It follows this pattern:
```bash 
cp OPTIONS SOURCE DESTINATION
```

- Recursive copy a directory with all subdirectories and files
```bash 
cp -r SOURCE DESTINATION
```
```bash 
cp -r mydir newcopy
```
