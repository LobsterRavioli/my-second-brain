# Daemon

A Daemon is a running process in background.
A daemon could be an http server, databases, on-demand cofiguration.
Many aspect from linux are effected from linux like firewalls and load balancing configuration
The daemon should be modifiable at runtime, so services can be stopped without reboot or shut down the machine.
These daemon could be controlled by a [[Service Manager]] or a script.
