# pstree

## Syntax
```bash
pstree
```

## Description
It shows processes in hierarchically way.
## Option

| Option | functionallity      |
| ------ | ------------------- |
| -p     | include anche i pid |


## Examples
- Mostra l'albero dei processi con i relativi pid.
```bash
pstree -p
```
