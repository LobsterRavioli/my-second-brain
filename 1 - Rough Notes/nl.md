# nl

The numbered line command can enumerate the lines of the file.

- enumerate lines
```bash
nl lollo.txt
```

- enumerate with 3 increments 
```bash
nl -i3 lollo.txt
```
- add suffix or postfix to enumeration
- 
- add tabulation to enumeration
