# renice

## Syntax
```bash
renice -10 -p 2164
```

## Description

## Option

| Option | functionallity                           |
| ------ | ---------------------------------------- |
| -g     | modify all processes to a specific group |
| -u     | modify all processes to a specific user  |


## Examples
- 
```bash
renice -10 -p 2164
2164 (process ID) old priority 0, new priority -10
```
