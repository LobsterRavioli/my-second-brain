# mkswap

## Syntax
```bash
mkswap device
```

## Description
Prepare a partition for usa as swap space. This command can also set up swap space in a file on another filesystem.

## Option

| Option | functionallity |
| ------ | -------------- |
|        |                |


## Examples
- Prepare a partition as swap space
```bash
mkswap /dev/hda5
```
