# rm

Utility used for remove files.
Syntax of rm:

```bash
rm OPTIONS FILE
```

- Remove a file
```bash
rm file1
```

- Secure removing, it will prompt you a confimation.
```bash
rm -i file1
```

- multiple removing
```bash
rm file1 file2 file3
```

- recursive deletion
```bash
rm -r dir
```

- recursive deletion with confirmation
```bash
rm -ri dir
```
