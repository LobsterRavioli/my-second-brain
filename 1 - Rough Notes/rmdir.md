# rmdir

rmdir is the utility for remove an empty directory.
- Removing a directory
```bash
rmdir dir1
```
- Removing multiple directory
```bash
rmdir dir1 dir2
```
- Remove dir1/dir2
```bash
rmdir -p dir1/dir2
```
