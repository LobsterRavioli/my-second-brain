# LD_LIBRARY_PATH

LD_LIBRARY_PATH is an environment path used for store temporary shared libraries.
Like:

```bash
LD_LIBRARY_PATH=/usr/local/mylib
```

To make changes permanent
```bash
export LD_LIBRARY_PATH=/usr/local/mylib
```
