# pgrep

## Syntax
```bash
pgrep [command of the file]
```

## Description
pgrep is searching for the [[Process ID]] of the process name.

## Option

| Option | functionallity |
| ------ | -------------- |
| -l     | long format    |


## Examples
- Search for process name
```bash
$ pgrep sleep
1201
```
